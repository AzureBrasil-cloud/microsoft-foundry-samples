# upload de arquivo

O Acess token obtido na etapa [1](./1.ACCESS_TOKEN_AQUISITION.MD)

```bash
export AZURE_PROJECT_URL="<URL_DO_PROJETO>"
```

```bash
FILEPATH=""
curl -X POST \
  "${AZURE_PROJECT_URL}/openai/files?api-version=2025-11-15-preview" \
  -H "Authorization: Bearer ${AZURE_ACCESS_TOKEN}" \
  -H "Accept: application/json" \
  -F "file=${FILEPATH};type=application/octet-stream" \
  -F "purpose=assistants"

```
