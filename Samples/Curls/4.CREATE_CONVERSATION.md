# Criação da Conversa

Este guia mostra como criar uma conversa usando o endpoint `/openai/conversations` com o Access Token obtido no [1.ACCESS_TOKEN_AQUISITION.MD](./1.ACCESS_TOKEN_AQUISITION.MD).

---

## Pré-requisitos
- Obter um Access Token (veja [1.ACCESS_TOKEN_AQUISITION.MD](./1.ACCESS_TOKEN_AQUISITION.MD)).
- Ter a variável `AZURE_PROJECT_URL` configurada (URL do recurso/projeto).
- Ter `curl` instalado no ambiente.

Exemplo de variável:
```bash
export AZURE_PROJECT_URL="https://<seu-recurso>.services.ai.azure.com/api/projects/<nome-do-projeto>"
export AZURE_ACCESS_TOKEN="<seu_access_token>"


```bash
curl -X POST "${AZURE_PROJECT_URL}/openai/conversations?api-version=2025-11-15-preview" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${AZURE_ACCESS_TOKEN}" \
  -d "{}"
```

```json
{
  "id": "<conversation_id>",
  "object": null,
  "created_at": null,
  "metadata": {}
}

```
