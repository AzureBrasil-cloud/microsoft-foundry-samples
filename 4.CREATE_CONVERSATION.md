# criação da conversation


```bash
curl -X POST "${AZURE_PROJECT_URL}/openai/conversations?api-version=2025-11-15-preview" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${AZURE_ACCESS_TOKEN}" \
  -d "{}"
```

```json
{
  "id": "<conversation_id>",
  "object": null,
  "created_at": null,
  "metadata": {}
}

```
